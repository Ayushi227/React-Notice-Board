# 📝 React Notice Board

A simple, interactive notice board web app built with **React** that allows users to view, add, and manage notices. This project demonstrates component-based UI design, state management, and basic CRUD functionality using React.

---

## 🚀 Features

- 🆕 Add new notices
- 🗑️ Delete notices
- 📋 View a list of posted notices
- 🧠 Uses React state for dynamic UI updates
- 📱 Responsive and user-friendly UI

---

## 🛠️ Tech Stack

- React (Create React App)
- JavaScript (ES6+)
- CSS for styling

## 💡 Highlights

✔ **Add Notice Functionality**  
Users can type notice text and add it to the board.

✔ **Interactive Notice List**  
Each notice renders on the board and updates automatically.

✔ **Delete Notices**  
Remove any notice with a single click.

✔ **React State Management**  
Uses React’s state hooks (`useState`) to manage notices dynamically.

## 🚀 How to Run Locally

1. **Clone the repository**
```bash
git clone https://github.com/Ayushi227/React-Notice-Board.git
cd React-Notice-Board
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm start
```

The app will open in your default browser at:
```
http://localhost:3000
```

---

## 🧠 How It Works

- The app uses **React components** to render the UI.
- Notices are stored in component state and updated using hooks.
- Adding a notice updates the list in real time without reloading.
- Deleting a notice removes it from the state and UI.
